D - Candy Distribution
A[l] + ... + A[r] = 0 (mod M)な(l,r)の個数を求める問題
実は考察ゲー

連続する要素の和なので累積和では？と考える
すると、B[r] - B[l-1] = 0 (mod M)な(l,r)の個数を求めればよい
これはB[l-1] = B[r] (mod M)というだけなので、結局個数を求めるには、
0 <= x < Mの各xに対してB[i] = x (mod M)なものの個数をnとすると、Σn(n-1)/2となる。
Mが大きいのでこれはハッシュを使ってカウントするとよい。

所感

d 400
iを起点として、連続した区間の個数を数える
n個あれば区間はn(n+1)/2個ある
全然違った、ダメ

累積和の配列をBとすると、
A[l~r]の和がMで割り切れる
⇔B[r]-B[l-1]がMで割り切れる
⇔B[r]≡B[l-1](mod M)
p:つまり累積和配列においてBi≡Bjとなるようなi,jを数えれば良い
これはMがでかすぎるので配列としては持てないがハッシュとしてなら持てるのでそれでよい
